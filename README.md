# Ideia geral:

1 - Utilizar uma interface para clicar em um determinado ponto na tela o mais rápido possível
2 - Utilizar uma base de dados de imagens para treinar escrita rápida

## 1: Clicker

Ao selecionar uma velocidade, o jogador deve clicar em círculos que aparecem aleatóriamente na tela.
Caso o tempo passe, o jogo é terminado e o score, guardado.
existe uma pontuação de score para cada dificuldade.
existe uma tabebla de placar com os 10 melhores placares.
